自研接口自动化测试平台

前言
作者很想吐槽一句，前后端分离的情况下，solo项目真心累，尤其是我这种前端渣渣，所以界面丑请多多包含！

环境
该项目测试信息管理与测试执行是分开的，可分开部署运行

JAVA 1.8
MYSQL
测试平台为war（Spring+SpringMVC+meavn）
测试执行为jar（Spring+meavn）

项目介绍

  • 环境管理
    • 增删查改测试环境地址

  • 接口管理
    • 增删查改接口API地址，并关联对应测试环境

  • 用例管理
    • 增删查改测试用例信息，包括关联的接口、请求头、请求方法、请求参数、断言信息、用例变量信息等，所有值可使用函数关键字进行取值，详细可看函数说明
    • 复制测试用例信息
    • 执行测试用例进行调试

  • 测试集管理
    • 增删查改测试集及里面相应的用例内容，包括用例信息、用例执行优先级、要求响应时间、赋值变量给该测试集里其他测试用例使用等等，变量值可使用函数关键字进行取值，详细可看函数说明
    • 复制测试集里面用例信息添加到另一个测试集
    • 执行测试集进行调试

  • 任务管理
    • 增删查改测试任务及里面相应的测试集，包括测试集执行优先级、任务定时执行时间设置、任务执行状态，任务执行结果列表及明细（执行器每10分钟执行一次）
    • 复制任务里面测试集信息添加到另一个任务

  • 函数助手：编写接口用例、赋值测试集变量时可调用的关键字
